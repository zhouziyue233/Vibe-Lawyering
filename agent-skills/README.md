# ⚡ 法律 AI 技能库 | Agent Skills

> 精选的法律领域 AI Agent 技能配置，覆盖律师、法务、研究者的核心工作场景。

---
## 如何找到技能 | How to find skills

每个技能以文件夹的形式包含在agent-skills文件夹下面，按照GitHub首页的名称和描述寻找对应的skills。


## 贡献说明 | How to Contribute

每个技能包含以下内容：

1. **技能名称与描述**：技能的名称和适用场景
2. **系统提示词（System Prompt）**：定义 AI 的角色、行为规范和工作流程
3. **使用示例**：输入与输出的示例

---

## 贡献你的技能 | Contribute a Skill

发现了一个有用的 AI 技能配置？欢迎提交 PR！
