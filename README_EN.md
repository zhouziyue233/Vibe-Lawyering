## 🌐 About

**Vibe-Lawyering** is an open-source resource aggregation platform for legal professionals — lawyers, legal researchers, in-house counsel, judges, prosecutors, and more. Our mission is to lower the barrier for the legal industry to adopt AI tools.

| Module | Content |
|--------|---------|
| ⚡ **Agent Skills** | Curated AI agent skill configurations for legal domains, ready to use with Claude, GPT, and more |
| 🗄️ **MCP Resources** | Legal database MCP tool lists and integration guides (Pkulaw, Westlaw, EUR-Lex, etc.) |
| 🎓 **AI & Coding Courses** | Zero-to-hero AI and programming courses tailored for lawyers |

## 🚀 Quick Start

### Using Agent Skills

1. Browse the [`agent-skills/`](agent-skills/README.md) directory to find the skill you need
2. Follow the instructions on the skill page to copy the prompt or install the config
3. Load the skill into your AI tool (Claude, GPT, Coze, etc.)

### Connecting Legal Database MCPs

1. Browse [`mcp-resources/`](mcp-resources/README.md) to see supported databases
2. Follow the tutorials in [`setup-guides/`](mcp-resources/setup-guides/) to complete integration
3. Combine Agent Skills with MCP tools for a unified "search + analyze" workflow

### Learning AI & Coding

1. Visit [`courses/`](courses/README.md) and choose a learning path
2. No technical background required — all content is grounded in legal work scenarios
