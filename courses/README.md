## 学习路径 | Learning Paths

### 🏃 快速入门路径（建议1-2周）

适合：想快速上手 AI 工具、提升工作效率的法律人

```
1. [AI 工具基础] 提示词技巧入门 (2h)
       ↓
2. [实战应用] 用 Claude 辅助合同审查 (3h)
       ↓
3. [工作流整合] 法律研究 AI 工作流搭建 (2h)
```

### 🎯 系统提升路径（建议1-3个月）

适合：希望深入掌握 AI 工具并具备基础编程能力的法律人

```
1. AI 工具全套课程（4门）
       ↓
2. Python 零基础入门 (8h)
       ↓
3. 法律文本数据处理 (6h)
       ↓
4. 构建你的第一个法律 AI 工具 (10h)
```

### 🔬 研究者路径（适合法学研究人员）

```
1. AI 辅助法律研究工作流
       ↓
2. 数据分析基础（Python/R）
       ↓
3. 法律数据可视化
       ↓
4. 实证法学研究方法与 AI 工具整合
```

## AI 应用课程 | AI Application Courses

### 入门级

#### 🤖 AI 工具入门：法律人的提示词技巧
- **时长：** 约 2 小时
- **难度：** ⭐ 零基础
- **语言：** 中文
- **费用：** 免费
- **内容概要：**
  - 什么是提示词（Prompt），为什么它很重要
  - 六大核心提示词技巧：角色扮演、分步骤、给例子、限制格式、迭代优化、思维链
  - 法律场景专项练习：合同审查提问、案例检索提问、法律咨询回复
- **详细内容：** [ai-for-lawyers.md#提示词技巧](ai-for-lawyers.md#提示词技巧)

#### 📋 用 Claude 辅助合同审查（实战）
- **时长：** 约 3 小时
- **难度：** ⭐⭐ 初级
- **语言：** 中文
- **费用：** 免费
- **内容概要：**
  - 合同审查 AI 工作流设计
  - 分段输入长合同的技巧
  - 风险条款识别提示词模板
  - 修改意见生成和格式化输出
  - 局限性：AI 不能替代专业法律判断
- **详细内容：** [ai-for-lawyers.md#合同审查实战](ai-for-lawyers.md#合同审查实战)

## 编程入门课程 | Coding Courses

### 零基础

#### 🐍 Python 零基础：处理法律文本数据
- **时长：** 约 8 小时
- **难度：** ⭐ 零基础
- **语言：** 中文
- **费用：** 免费
- **你将学到：**
  - Python 基础语法（变量、循环、函数）
  - 读取和处理 Word 文档、PDF
  - 文本搜索和批量处理
  - 用 Python 整理案件数据
- **推荐资源：**
  - [廖雪峰 Python 教程](https://liaoxuefeng.com/books/python/) `[中文][免费]`
  - [Python for Lawyers（GitHub）](https://github.com/pythonforlawyerss) `[EN][免费]`
- **详细课程：** [coding-for-lawyers.md#python入门](coding-for-lawyers.md#python入门)

#### 📄 用代码批量处理合同
- **时长：** 约 4 小时
- **难度：** ⭐⭐ 初级（需先完成 Python 入门）
- **语言：** 中文
- **费用：** 免费
- **你将学到：**
  - 批量读取合同文件夹
  - 提取合同关键信息（金额、日期、当事人）
  - 生成合同台账 Excel 表格
  - 检测缺失条款
- **详细课程：** [coding-for-lawyers.md#批量处理合同](coding-for-lawyers.md#批量处理合同)

### 初中级

#### 📊 数据可视化：让法律数据讲故事
- **时长：** 约 6 小时
- **难度：** ⭐⭐ 初中级
- **语言：** 中文/英文均有资源
- **费用：** 免费
- **你将学到：**
  - 案件数据的统计分析
  - 用 Python 绘制胜诉率分析图
  - 裁判文书地区分布热力图
  - 法条引用频率分析
- **推荐资源：**
  - [Matplotlib 中文文档](https://matplotlib.net/) `[中文][免费]`
  - [Seaborn 可视化教程](https://seaborn.pydata.org/tutorial.html) `[EN][免费]`
- **详细课程：** [coding-for-lawyers.md#数据可视化](coding-for-lawyers.md#数据可视化)

#### 🛠️ 构建你的第一个法律 AI 工具
- **时长：** 约 10 小时
- **难度：** ⭐⭐⭐ 中级
- **语言：** 中文
- **费用：** 免费（需 Claude/OpenAI API 费用）
- **你将学到：**
  - 调用 Claude API 处理法律文本
  - 构建合同批量审查工具
  - 搭建简单的法律问答界面
  - 部署到本地或云端
- **详细课程：** [coding-for-lawyers.md#构建法律工具](coding-for-lawyers.md#构建法律工具)
