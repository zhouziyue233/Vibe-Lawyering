# 中国法律数据库接入指南

> 本文档详细介绍各主流中国法律数据库的基本信息、覆盖范围和 MCP 接入配置。

---

## 免费数据库

### 国家法律法规数据库（flk.npc.gov.cn）

**主办单位：** 全国人民代表大会常务委员会办公厅
**覆盖范围：** 现行有效的法律、行政法规、地方性法规、司法解释、规范性文件
**数据更新：** 实时更新
**语言：** 中文（部分提供英文版）
**访问：** 完全免费，无需注册

**特点：**
- 权威性最高，为官方正式发布渠道
- 支持全文检索
- 提供法律修订历史

**MCP 配置：**
```json
{
  "mcpServers": {
    "npc-legal": {
      "command": "npx",
      "args": ["-y", "@vibe-lawyering/npc-legal-mcp"],
      "env": {
        "BASE_URL": "https://flk.npc.gov.cn/api"
      }
    }
  }
}
```

---

### 中国裁判文书网（wenshu.court.gov.cn）

**主办单位：** 最高人民法院
**覆盖范围：** 全国各级法院裁判文书（含民事、刑事、行政、执行）
**数据时间：** 2013年至2023年底（目前已停止更新，移至"人民法院案例库"）
**语言：** 中文
**访问：** 免费，需注册账号

**特点：**
- 数据量最大，超过1亿份文书
- 支持多维度筛选（地区、年份、案由、法院层级等）
- 可下载全文

**注意事项：**
- 2024年起最高院将案例公开平台迁移至"人民法院案例库"
- 部分文书因涉及隐私已屏蔽或删除

---

### 人民法院案例库（案例.court.gov.cn）

**主办单位：** 最高人民法院
**覆盖范围：** 参考性案例、典型案例（精选，非全量）
**上线时间：** 2024年2月
**语言：** 中文
**访问：** 免费，需注册

**特点：**
- 替代裁判文书网的官方新平台
- 案例经过筛选，质量较高
- 提供案例要旨和适用规则

---

## 商业数据库

### 北大法宝（pkulaw.com）

**运营方：** 北京大学法制信息中心 / 北大英华科技有限公司
**覆盖范围：**
- 法律法规（国内最全，逾千万条）
- 司法案例（含裁判文书网全量数据）
- 法律期刊（600余种）
- 法律英译（双语法规）
- 合同范本、律师文书模板

**订阅方式：** 年度订阅，个人/机构版本不同定价
**MCP 状态：** 🟡 开发中（需官方 API 授权）

**适合人群：** 律师事务所、法学院校、企业法务

---

### 威科先行（wkinfo.com.cn）

**运营方：** Wolters Kluwer（威科集团）中国
**覆盖范围：**
- 法律法规（含地方性法规）
- 司法案例
- 实务分析（专家点评）
- 税务、劳动、公司法专项数据库

**订阅方式：** 年度订阅，模块化定价
**MCP 状态：** 🟡 开发中

**特点：** 实务分析内容质量高，有专业律师/专家注释

---

### 法信（faxin.cn）

**运营方：** 人民法院出版社
**覆盖范围：**
- 最高人民法院指导案例
- 司法解释、审判参考
- 裁判规则梳理

**背景：** 具有官方背景，内容权威
**MCP 状态：** 🟡 开发中

## 数据库选型建议

| 需求 | 推荐数据库 |
|------|-----------|
| 查找现行法律法规原文 | 国家法律法规数据库（免费、权威） |
| 检索裁判文书（历史） | 中国裁判文书网 |
| 检索高质量参考案例 | 人民法院案例库 |
| 综合法律研究（全量） | 北大法宝 |
| 实务分析和专家意见 | 威科先行 |
| 最高院裁判规则研究 | 法信 |
